# Features
* Add quote marks to text (line)
* Add quote marks to text (space)

# Prerequisites
* [Node.js](https://nodejs.org/en/)

# Running
### 1. Build
````
# Install vsce
npm install -g vsce

# Package
vsce package

# Get your version
qmarks-[VERSION].vsix
````

### 2. Run
````
# Install dependencies
npm install

# Start
F5 to run.
````
